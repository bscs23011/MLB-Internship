What is Deep Learning?
Deep Learning is a subset of Machine Learning that uses Artificial Neural Networks with multiple hidden layers to automatically 
learn patterns and features from large amounts of data. deep learning can perform feature extraction on its own,
making it highly effective for complex tasks such as image recognition, speech recognition, natural language processing, and autonomous driving.

Difference between Machine Learning and Deep Learning.

Deep Learning is a specialized subset of Machine Learning . While all deep learning is machine learning, 
traditional ML relies on structured data and human intervention to define features,
whereas DL utilizes multi-layered neural networks to automatically extract features from big unstructured datasets

What is a Perceptron?
A Perceptron is the simplest unit of an Artificial Neural Network and acts as a single artificial neuron.
It receives one or more input values, multiplies each input by a weight, adds a bias, applies an activation function, and produces an output.

Which activation functions you explored and where they are commonly used.
ReLU (Rectified Linear Unit)
Output range: 0 to infinity, Most commonly used in hidden layers of deep neural networks.
Fast computation. Helps reduce the vanishing gradient problem. Improves training speed.

Sigmoid
Output range: 0 to 1. Commonly used in the output layer of binary classification problems.
Produces probability values.
Can suffer from the vanishing gradient problem.

Tanh 
Output range: −1 to 1. Often used in hidden layers, especially in some recurrent neural networks (RNNs).
Zero-centered output.
Can also experience the vanishing gradient problem in deep networks.
