What is Classification
Classification is a supervised machine learning technique used to predict categorical labels or classes instead of numerical values.

Difference between Regression and Classification.
Regression can give and predict a range of numerical values while classification just predict the category or the lable.

Evaluation metrics used.
Accuracy: Measures the percentage of correctly classified samples.
Precision: Measures how many predicted class labels were actually correct.
Recall: Measures how many actual class instances were correctly identified.
F1-Score: The harmonic mean of Precision and Recall providing a balanced performance measure.
Confusion Matrix: Shows the number of correct and incorrect predictions for each class making it easier to identify where the model made mistakes.

Model performance and observations. 
Logistic Regression model was trained on the Iris dataset after splitting the data into training and testing sets and standardizing the features
A Decision Tree Classifier was also trained to compare its performance with Logistic Regression.
Both models achieved similar high accuracy, precision recall and f1 score indicating that the Iris dataset is well suited for classification tasks
